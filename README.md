Sistema para gerenciamento de clínica veterinária

 

Funcionalidades
1 Cadastrar clientes,
2 atualizar dados,
3 marcar, remarcar ou desmarcar consultas,
4 enviar relatório médico

 

Usuários:
5 Auto cadastro,
6 Verificar/ atualizar dados cadastrais,
7 Agendar consultas
8 Desmarcar consultas
